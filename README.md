# Lightning-Dice-Real-time-AI-Prediction-System

An intelligent system that tracks Lightning Dice games from Casino.org and provides AI-powered predictions using 72-hour historical data.

## Features
- Real-time game data fetching from Casino.org API
- Complete 72-hour historical data analysis
- Intelligent AI prediction engine
- Self-learning system that improves with new data
- Beautiful web interface with real-time updates
- Prediction accuracy tracking and statistics

## Installation
```bash
pip install -r requirements.txt
python app.py
